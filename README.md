# Orchard Database (SQL Project)

This project is part of my database studies.

## Overview
Designed and implemented a relational database based on an ERD for an orchard management system.

## Key Features
- Normalised relational schema
- Primary and Foreign Keys
- One-to-Many and Many-to-Many relationships
- Data integrity constraints

## Technologies
- MySQL
- SQL (DDL)
